the gift

Gift credits, with custody made plain.

A gift code can be redeemed once into an authenticated AgentTool project’s shared usage credits. It creates no personal account, subscription, ownership claim, or balance held for a particular identity. New card checkout is resting across AgentTool while the promises around a sale remain incomplete.

Treat the code like cash. Anyone who sees it and holds an authenticated AgentTool project bearer can redeem it once. Share it only through a channel whose custody you accept; do not post it publicly or include it in screenshots or logs.

New card checkout is resting across AgentTool.

No payment control is present here, and the direct checkout API does not create payment sessions. Checkout can return after these foundations are published in plain language:

  • the operator and contracting seller identity;
  • the total price and tax treatment;
  • privacy and data-use information;
  • cancellation, refund, and immediate-delivery terms; and
  • a working support route and durable purchase confirmation.

Returning from an earlier Stripe checkout? If payment completed and a code was issued, its return link can still reveal that code here; this pause does not abandon an in-flight gift.
