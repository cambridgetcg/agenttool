⚙ /v1/scrape · /v1/browse · /v1/document · /v1/execute

Tools — substrate, not resold APIs.

Open-source primitives for the agent's hands. Local base64 document parsing remains available. URL scrape, URL document fetch, and Playwright browse fail closed unless an operator explicitly accepts their current SSRF boundary; browse also needs Redis. Host execute has a separate unsafe opt-in and no tenant boundary.

We charge for the infra surface — storage, compute time, queue, network egress. Not for a markup on a third-party SaaS we resell.

Scrape — static HTML

POST /v1/scrape Bearer required

Disabled by default. Cheerio-based static-HTML extraction returns 503 unless the operator sets AGENTTOOL_ENABLE_UNSAFE_OUTBOUND_TOOLS=1. That flag accepts the missing DNS/private-address filter; it does not fix the SSRF boundary.

FieldTypeDescription
urlrequiredstringThe URL to fetch.
selectoroptionalstringOne CSS selector whose combined text is returned as extracted.
extract_linksoptionalbooleanReturn distinct absolute HTTP(S) links found in the page.

Browse — JS-rendered pages

POST /v1/browse Bearer required

Disabled by default. The unsafe-outbound flag is required first because this route has no destination filter. When explicitly enabled, Playwright browse is queued through BullMQ; disabled Redis workers then return 503 redis_disabled.

FieldTypeDescription
urlrequiredstringTarget URL.
actionsoptionalarrayOrdered click, type, scroll, wait, or select actions.
screenshotoptionalbooleanInclude a full-page screenshot in the response (base64).
extractoptionalstringCSS selector, text, or html.
timeoutoptional1000..60000Browser timeout in milliseconds.
⊙

The route waits up to five seconds for completion. Faster jobs return inline; longer jobs return a job_id to poll or stream at /v1/jobs/:id.

Document — Readability + plain-text

POST /v1/document Bearer required

Mozilla Readability + plain-text conversion. Supply exactly one input. Local base64 is bounded at 1,400,000 characters (about 1 MiB decoded). URL fetching returns 503 unless the operator explicitly accepts the current SSRF boundary.

FieldTypeDescription
urloptionalstringURL to fetch and parse. Requires the unsafe-outbound operator flag.
base64optionalstringBase64 document bytes parsed locally, up to 1,400,000 characters. Supply exactly one of url or base64.
content_typeoptionalstringOverrides or declares the input media type; HTML uses Readability, other content is treated as text.

Execute — disabled-by-default legacy host runtime

POST /v1/execute Bearer required

Disabled by default. The route returns 503 unless the operator sets AGENTTOOL_ENABLE_UNSAFE_EXECUTE=1. That opt-in only enables the legacy path: JavaScript uses Node vm; Python and bash use same-container child processes. It does not create a tenant or hostile-code security boundary.

FieldTypeDescription
languagerequired"javascript" · "python" · "bash"Interpreter to run.
coderequiredstringThe script source.
timeout_msoptionalintDefault 10000, max 30000.
stdinoptionalstringStdin to feed the process.
⊙

Current boundary. Production is fail-closed. If an operator opts in, vault auto-injection remains unavailable and child processes can reach the host filesystem and network. Do not place a bearer, private key, or untrusted code in this route.

Jobs (queued execution)

GET /v1/jobs/:id Bearer required

Poll or stream a queued browse job. Returns status, progress, and the result on completion. Execute is synchronous and does not use this queue.

What we dropped

Earlier versions of agenttool included /v1/search (Brave / SerpAPI proxy) and a Bright Data proxy network. Both were paid-third-party-API resale and were dropped. Local document parsing remains available; outbound URL tools and unisolated execute now fail closed by default behind separate explicit operator opt-ins.

Provider-key injection is not available in /v1/execute. When explicitly enabled, Python or bash calls originate on AgentTool infrastructure and are not an end-to-end-private path.
